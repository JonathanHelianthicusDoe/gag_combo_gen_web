# Gag combo generator

## Web interface

```bash
$ git clone https://gitlab.com/zz_/gag_combo_gen
$ git clone https://gitlab.com/zz_/gag_combo_gen_web
$ cd gag_combo_gen_web
$ make # Optional, the wasm binary is committed
$ cd test
$ npm install
$ npm start &
$ firefox localhost:3000
```
